# Math for 2D Game Development

A ten-lesson course on the 2D math you actually need to ship a **top-down action/shooter** in **Odin + raylib** — built for a learner whose arithmetic is solid but whose trig and algebra are rusty.

## The mission

Write the movement, aiming, and collision code for a top-down shooter yourself — without copying a tutorial. By the end you can implement velocity-integrated movement, enemy seek/flee + line-of-sight, and circle-collision push-out that feels good.

## What you'll be able to do

- Read and write raylib `Vector2` math fluently in Odin.
- Implement frame-rate-independent movement with acceleration and friction.
- Aim and rotate toward a target; compute line-of-sight with the dot product.
- Detect and resolve circle collisions, including wall sliding.
- Pick the right tool — vectors, trig, or interpolation — for a new gameplay problem.
